Spring Session实现分布式Session共享

发表于 2019-12-05 | 分类于 分布式
字数统计: 1,177 | 阅读时长 ≈ 6

Session是服务端用来存储用户和服务器之间的会话信息。单机情况下，不存在Session的共享问题。但是分布式情况下，Session如果不共享那当不同请求请求到不同机器时就会出现问题。本篇博客就介绍Spring Session实现Session的共享。
