遍历文件夹

for循环即可遍历文件夹

命令：for [参数] %%变量名 in (匹配符) do (执行的命令)

参数可以分为/d,/r,/l,/f,无参

  • 无参 : 遍历当前文件夹下的文件
  • /d : 遍历当前文件夹下的子文件夹
  • /r : 深度遍历当前文件夹下的所有文件
  • /l : 迭代数值范围，使用迭代变量设置起始值，然后逐步执行一组范围的值，直到该值超过所设置的终止值
  • /f : 用于解析文件内容
