Windows下bat使用小技巧

发表于 2019-04-21 | 分类于 Windows
字数统计: 238 | 阅读时长 ≈ 1

最近在整理一些软件在Windows下的安装，然后需要用到bat的一些语法，就记录一下，掌握一下确实方便了很多。

1、在一个bat中打开多个窗口

Zookeeper集群需要打开同时打开三个Zookeeper，以下就能一个bat同时打开三个了。

cmd /k会打开一个新窗口后执行指令，且原窗口不会关闭。

2、命令输出日志

使用bat执行某命令时发生闪退，然后因为没有日志文件，所以不清除发生了什么错误，这时可以使用 命令 >> 输出文件，输出命令执行后的信息。
